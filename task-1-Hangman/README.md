# Task 1 - Hangman Game

## Description
This is a simple Hangman game built using Python where the user guesses a word letter by letter.

## Features
- Random word selection
- Limited attempts
- User input handling
- Predefined words

## How to Run
1. Open terminal
2. Navigate to folder
3. Run:
   python main.py
